## 🛠️ Local Development

Clone the repository and install dependencies:

```bash
git clone https://github.com/Dipalikar/DipaliKar.git
cd DipaliKar
npm install
```

Start the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
